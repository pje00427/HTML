테두리 관련 스타일

        선택자{
            * 선모양
            border-style:none|dotted|dashed|solid|double|groove|inset|outset|ridge;
            * 테두리 두께
            border[-위치]-width : 테두리 두께;

            * 테두리 색상
            border[-위치]-color :테두시 색상;

            * 한꺼번에
            border[-위치]:두께 스타일 색상;

            * 테두리를 둥그스름하게
            border[-위치]-radius:숫자 숫자;

            * 테두리에 그림자 효과
            box-shdow:가로거리 세로거리 흐림정도 번짐정도 색상
        }
