CSS 기본 선택자

선택자란 ?
특정 HTML요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택해서 원하는 스타일과 기능을 적용시킬 수 있음

1.모든 선택자 (*)

해당 이 문서상의 모든 요소들을 선택할 때 사용

        * {
            스타일속성:값;
            스타일속성:값;
        }

2. 태그 선택자(태그명)

해당 이문서내에 해당 같은 태그들 모두 선택할 때 사용

        태그명{
            스타일속성:값;
            스타일속성:값;

        }

3.아이디 선택자 (#아이디명)

해당 이 문서내에 특정 HTML 요소 하나만을 선택하고자 할 때

        #아이디명{
            스타일속성:값;
            스타일속성:값;
        }

4. 클래스 선택자 (.클래스명)

해당 이 문서내에 내가 원하는 여러개의 요소를 선택하고자 할 때 사용

        .클래스명{
            스타일속성:값;
        }
