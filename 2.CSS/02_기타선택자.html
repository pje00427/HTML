선택자 뒤에[]를 사용해서 속성과 어떤 속성값을 사용하는 방식

        선택자[속성=|~=||=|^=|$=|*=속성값]{
            스타일 속성:값
        }

2. 자손 선택자와 후손 선택자

자손 : 바로하위요소들 / 후손: 하위요소들 전부

        * 자손선택자
        a>b{
            스타일속성:값
        }
        * 후손선택자
        a b{
            스타일 속성:값

3. 동위 선택자

동위 관계(같은레벨)에 있는 뒤에 위치한 특정 요소를 선택할 때 사용

        * a요소 뒤에 있는 b요소 하나만을 선택
        a+b{
            스타일속성:값;
        }
        a~b{
            스타일속성:값;
        }
