배경 관련 스타일

background-color : 배경색 지정하고자 할 때 사용

        선택자{
            background-color:색상명|rgb|rgba|hsl|hsla|16진수;
        }

background-clip : 배경을 적용시키고자 하는 범위를 지정할 때 사용

        선택자{
            background-clip:border-box|padding-box|content-box
        }

배경 이미지 테스트

배경색 뿐만 아니라 배경 이미지를 넣을 수도 있다.

        선택자{
            background-image:url("이미지 경로");
            background-repeat:repeat|repeat-x|repeat-y|no-repeat;
            background-size:auto|contain|cover|px px|% %;
            background-position:left,right,center top,bottom,center|px px| % %;
            background-attachment:scroll|fixed;
        }
