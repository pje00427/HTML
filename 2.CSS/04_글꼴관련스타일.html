font-size 스타일 속성 : 텍스트이 크기를 변경 할 때 사용

선택자{font-size:크기(px|em|%);}
  • 고정크기 px테스트
  • 가변크기 em테스트
  • 가변크기 %테스트

font-weight 스타일속성 : 텍스트의 굵기를 표현할 때 사용

        선택자{
            font-weight : normal|bold|bolder
        }
  • 원래굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-variant 스타일속성 : 텍스트문구를 작은 대문자로 변경할 때 사용

선택자{font-variant:small-caps}

font-style 스타일속성 : 텍스트문구를 기울이고자 할 때 사용

선택자{font-style:nomal|italic|oblique}
  • italic 기울임 글꼴
  • oblique 원래글자를 기울인셈

font 스타일 속성 : 글꼴과 관련된 스타일을 한번에 지정할 수 있음

        선택자{
            font:font-style font-variant font-weight font-size/line-height
        }
