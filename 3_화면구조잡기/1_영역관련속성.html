영역관련 속성

1)content 영역 : 요소 내의 내용영역
2)border 영역 : 요소의 테두리 영역
3)padding 영역 : content와 border사이의 영역(여백)
4)margin 영역 : 타 요소들간의 간격 영역 (바깥)
div의 테두리 까지 포함한 가로길이
= width + 양쪽 padding + 양쪽 border
= 300px 20px 40px
= 360px
div의 테두리까지 포함한 세로길이
= height + 상하padding + 상하 border
= 150px 20px 40px
= 210px
